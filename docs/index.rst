Welcome to hash-ring-ctypes's documentation!
============================================

``hash-ring-ctypes`` is a fast ctypes-based wrapper around
`libhashring <https://github.com/chrismoos/hash-ring/>`_.

This library is available on `GitHub <https://github.com/mattdennewitz/hash-ring-ctypes/>`_
under the `MIT license <https://github.com/mattdennewitz/hash-ring-ctypes/blob/master/LICENSE>`_.
